nihongo.io
=========

Open source Japanese Dictionary written in Go: [http://nihongo.io](http://nihongo.io)

### How to run:
1. `git clone https://github.com/gojp/nihongo.git`
2. Install dependencies: `go get ./...`
3. Run the app: `godep go run main.go`
